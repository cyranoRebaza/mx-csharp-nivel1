# C# Nivel 1 – Prácticas y Apuntes

Repositorio de estudio y práctica del curso **C# Nivel 1** dictado por Maximiliano Sar Fernández (MaxiPrograma).

Este repositorio documenta de forma ordenada el aprendizaje de los fundamentos de C#, incluyendo teoría,
ejercicios prácticos y buenas prácticas de organización, versionado y documentación.

---

## Tecnologías utilizadas

- C#
- .NET 9
- Visual Studio Code / Visual Studio
- Git y GitHub

---

## Objetivo del repositorio

- Comprender los fundamentos del lenguaje C#
- Practicar lógica de programación con aplicaciones de consola
- Desarrollar hábitos profesionales de organización de proyectos
- Documentar el proceso de aprendizaje de forma clara y estructurada
- Servir como material de repaso y referencia futura

---

## Organización del repositorio

```txt
mx-csharp-nivel1/
├── docs/              # Guías técnicas (setup, herramientas, workflow)
├── 01-teoria/         # Teoría y apuntes organizados por unidad
├── 02-ejercicios/     # Ejercicios prácticos (un proyecto por ejercicio)
├── README.md
└── .gitignore

```

---

## Flujo de trabajo (Workflow)

El estudio y la práctica siguen el siguiente flujo:

1. Leer la teoría de cada unidad en `01-teoria/unidad-XX`
2. Tomar apuntes personales en `notas.md`
3. Resolver los ejercicios prácticos en `02-ejercicios/unidad-XX`
4. Crear un proyecto de consola por ejercicio
5. Realizar commits incrementales por ejercicio completado
